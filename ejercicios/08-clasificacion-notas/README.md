# Ejercicio 08 - Clasificación de notas

Este ejercicio permite clasificar una nota numérica (0-20) en diferentes categorías de calificación.

## Descripción
El programa solicita al usuario una nota entre 0 y 20, y la clasifica en las siguientes categorías:
- Muy bueno: 16-20
- Bueno: 14-15.99
- Lo suficientemente bueno: 12-13.99
- Medio: 10-11.99
- Insuficiente: < 10
