# Ejercicio 10 - Analizar contenido de un directorio

Este ejercicio analiza el contenido de un directorio y proporciona información sobre los archivos y subdirectorios.

## Descripción
El programa analiza un directorio (pasado como parámetro) y cuenta:
- Archivos estándar
- Subdirectorios
- Archivos ejecutables

## Uso
```
./analizar_directorio.sh /ruta/del/directorio
```

## Archivos
- `scripts-sencillos/`: Versiones simplificadas de los scripts
- `scripts-tal-cual/`: Versiones completas con características adicionales
