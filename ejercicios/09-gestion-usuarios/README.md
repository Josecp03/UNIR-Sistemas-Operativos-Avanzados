# Ejercicio 09 - Menu de gestión de usuarios

Este ejercicio proporciona un menú interactivo para gestionar y consultar información de usuarios del sistema.

## Descripción
El programa presenta un menú que permite:
1. Verificar si un usuario existe en el sistema
2. Obtener el UID (SID en Windows) de un usuario
3. Salir del programa

## Archivos
- `scripts-sencillos/`: Versiones simplificadas de los scripts
- `scripts-tal-cual/`: Versiones completas con características adicionales
